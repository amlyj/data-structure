# 希尔排序

### JavaScript实现
```js
var arr = [49, 38, 65, 97, 76, 13, 27, 49, 55, 04], len = arr.length;

for (var index = Math.floor(len / 2); index > 0; index = Math.floor(index / 2)) {
    for (var i = index; i < len; i++) {
        for (var j = i - index; j >= 0 && arr[j] > arr[index + j]; j -= index) {
            var temp = arr[j];
            arr[j] = arr[index + j];
            arr[index + j] = temp;
        }
    }
}
console.log(arr);
```
